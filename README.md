# Vrinda-Store-Data-Analysis
* Performed end-to-end data analysis on Vrinda Store’s 2022 sales dataset using Excel to generate actionable business insights.
* Cleaned and preprocessed data by handling missing values, correcting inconsistencies, and standardizing categories.
* Engineered features such as Age Groups and Month to enable time-based and demographic analysis.
* Analyzed sales trends, order volume, customer behavior, and order status using Pivot Tables and Charts
* Identified key patterns including gender-based purchasing trends and top-performing states.
* Developed an interactive Excel dashboard with integrated charts and KPI metrics.
* Implemented slicers (Month, Channel, Category) for dynamic filtering and real-time data exploration.
* Derived insights on high-value customer segments (women aged 30–49) and recommended targeted strategies to boost sales.
